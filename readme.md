## Inststruction to run the api server locally

Install yarn package manager:

```console
$ npm install -g yarn
```

Rename config file inside config folder from config.sample.env to config.env and

- MONGO_URI
- API_SERVER_ENDPOINT
- JWT_SECRET

Install project dependecies

```console
$ yarn
```

To Run the server

```console
$ yarn start
```

Server will run
